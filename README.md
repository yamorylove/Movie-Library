##Описание программы

Movie Library - это GUI-приложение для ведения личной коллекции фильмов. Приложение позволяет добавлять фильмы, фильтровать их по жанру и году выпуска, сохранять и загружать данные в формате JSON, а также удалять отдельные записи.
Возможности:

Добавление фильмов с указанием названия, жанра, года выпуска и рейтинга


Фильтрация фильмов по жанру (частичное совпадение)


Фильтрация по году выпуска (точное совпадение)


Сохранение данных в JSON-файл


Автоматическая загрузка данных из JSON-файла при запуске


Удаление выбранного фильма


Валидация вводимых данных (проверка пустых полей, год должен быть числом, рейтинг от 0 до 10)

Инструкция по запуску
Требования:

Python 3.6 или выше


Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

Запуск:

Откройте терминал или командную строку


Перейдите в директорию проекта:

bash

cd movie_library


Запустите приложение:

bash

python movie_library.py

Примеры использования
Добавление фильма:

Заполните все поля формы:


Название фильма: "Побег из Шоушенка"


Жанр: "Драма"


Год выпуска: 1994


Рейтинг: 9.3


Нажмите кнопку "Добавить фильм"


Фильм появится в таблице в нижней части окна

Фильтрация:

По жанру: Введите текст в поле "Фильтр по жанру" (например, "Драма") и нажмите "Применить фильтр" - будут показаны фильмы, содержащие этот текст в жанре


По году: Введите год в поле "Фильтр по году" (например, 1994) и нажмите "Применить фильтр"


Можно комбинировать оба фильтра одновременно


Для сброса нажмите "Сбросить фильтр"

Сохранение и загрузка:

Данные сохраняются автоматически в файл movies.json при каждом добавлении или удалении фильма


При запуске приложения данные автоматически загружаются из файла movies.json


При первом запуске создаётся файл с примерами фильмов

Удаление:

Выберите фильм в таблице (щёлкните по строке) и нажмите "Удалить выбранный фильм"


Подтвердите удаление во всплывающем окне

После добавления своих фильмов попробуйте:

Отфильтровать фильмы по жанру "Криминал"


Найти все фильмы 1994 года выпуска


Добавить фильм с рейтингом 11 (получите сообщение об ошибке)


Закрыть и снова открыть приложение, чтобы проверить сохранение данных

Валидация данных
Приложение проверяет:

Название и жанр не могут быть пустыми


Год выпуска должен быть целым числом в диапазоне от 1888 до 2026


Рейтинг должен быть числом в диапазоне от 0 до 10


При удалении необходимо выбрать фильм из таблицы
